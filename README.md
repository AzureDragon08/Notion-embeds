# Notion-embeds
>A niche embed for your notion Program tracking
# USE
1. create a new Embed block in your Notion doc.
2. set embeded url to [https://ghchart.rshah.org/{USERNAME}](URL) replacing [{USERNAME}](URL) with your Git Hub Username.
3. adjust the size of embed.
